# Architecture

This directory contains the evolving architectural definition of the Forever Home.

Use it for:

- floor plans and schematic options
- exterior elevations and building sections
- massing studies
- room adjacencies and circulation diagrams
- Revit model notes and exports
- architect and engineer review comments

Files in this directory are conceptual owner-development materials unless explicitly marked otherwise. They are not permit-ready or engineered construction documents.
