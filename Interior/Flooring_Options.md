# Flooring Options

**Status:** Owner decision support — carpet is **not** assumed whole-house  
**Related:** [Kitchen_Dining_Porch_Specs.md](../Architecture/Kitchen_Dining_Porch_Specs.md), [Stairs_Half_Story.md](../Architecture/Stairs_Half_Story.md), [Living_Room.md](Living_Room.md)  
**Aesthetic filter:** Warm, traditional, welcoming grandma home — **avoid cool gray floors**

The user is **unsure about carpet**. This matrix recommends finishes by zone, with carpet used only where it earns its keep (stairs baseline, optional soft zones).

---

## 1. Decision summary (recommended package)

| Zone | Recommended finish | Carpet? |
|---|---|---|
| Kitchen | Engineered hardwood **or** LVP (wet-tolerant) | **No** wall-to-wall |
| Dining | Same continuous material as kitchen | **No** wall-to-wall; optional **area rug** under table |
| Living room | Engineered hardwood + **area rug** under loveseat/recliner | **No** wall-to-wall preferred |
| Three-season porch | Porcelain wood-look tile **or** composite decking | **No** |
| Half-story stairs | **Stair-rated carpet** (baseline) **or** wood treads + runner | Soft underfoot preferred |
| Private suite bedrooms | Carpet **or** hardwood + rugs | Owner choice |
| Lower level | LVP or carpet tile (practical) | Optional |

**Core idea:** Hard, warm floors for open social spaces (easy cleaning, continuous open-concept look); soft goods as **rugs and stair covering** rather than wall-to-wall carpet everywhere.

## 2. Whole-house candidates

### Option A — Engineered hardwood (primary recommendation for main living)

| Attribute | Spec |
|---|---|
| Product type | Engineered hardwood, oak or hickory wear layer |
| Plank width | **5"–7"** |
| Finish | Matte or low-sheen urethane; wire-brushed optional |
| Color | Warm natural, honey, or light amber — **not gray washed** |
| Thickness | **1/2"–5/8"** typical (confirm transitions) |
| Install | Float, glue, or nail/staple per product; follow underlayment for comfort/sound |
| Best in | Living, dining, kitchen (with care), hall, suite if desired |
| Pros | Authentic grandma warmth, ages with house, high-end feel |
| Cons | Water sensitivity vs LVP; cost; kitchen/porch caution |

**Kitchen use:** Allowed if seams sealed, mats at sink/range, and spills wiped promptly. Many grandma homes use wood in kitchen successfully.

### Option B — Luxury vinyl plank (LVP) (practical alternate)

| Attribute | Spec |
|---|---|
| Product type | Rigid-core LVP, waterproof rating |
| Plank width | **7"–9"** wood-look |
| Wear layer | **20 mil+** residential heavy |
| Color | Warm oak photo/realistic grain |
| Best in | Kitchen, dining, lower level, baths (if rated), high-spill zones |
| Pros | Water resistant, easier DIY repair, softer underfoot than tile, cost-flexible |
| Cons | Less “heirloom” than real wood; quality varies widely |

**Use when:** Prioritizing spill resistance and budget while keeping wood look continuous kitchen→dining.

### Option C — Hardwood + LVP hybrid

| Zone | Material |
|---|---|
| Living + dining | Engineered hardwood |
| Kitchen | LVP matched in tone **or** same hardwood |
| Porch | Tile / composite |
| Stairs | Carpet or wood + runner |

Match **undertone** (warm yellow/red oak family) so transitions feel intentional. Use **reducing thresholds** only where thickness demands.

### Option D — Wall-to-wall carpet (use sparingly)

| Attribute | Spec |
|---|---|
| Fiber | Nylon or wool-blend, dense |
| Pile | Low cut pile or patterned loop |
| Pad | Quality 6–8 lb rebond or rubber (not overly thick) |
| Color | Warm neutrals, soft patterns |
| Best in | Bedrooms, optional living **if** owner insists; **stairs baseline** |
| Avoid as only finish in | Kitchen, porch, mudroom, unconditioned porch |

## 3. Zone-by-zone matrix

| Zone | 1st choice | 2nd choice | Avoid | Soft layer |
|---|---|---|---|---|
| **Kitchen** | Eng. hardwood A **or** LVP B | Tile (warmer porcelain wood-look) | Deep carpet; cool gray tile | Washable runner at sink |
| **Dining** | Continuous with kitchen | Same | Wall-to-wall carpet | Optional 5×8 / 6×9 area rug under table (low profile) |
| **Living** | Eng. hardwood + rug | LVP + rug | Empty hard floor only (cold feeling) | Area rug under loveseat + recliner |
| **3-season porch** | Porcelain wood-look tile | Composite deck boards | Carpet; interior hardwood | Outdoor rug optional, UV/wet rated |
| **Stairs (half-story)** | Stair carpet full wrap | Wood treads + **carpet runner** | Slippery polished wood no rail grip | Soft is the point |
| **Suite bedrooms** | Carpet **or** hardwood + rugs | LVP | Commercial hard only | Rugs if hard surface |
| **Mudroom / laundry** | LVP or tile | — | Carpet | — |
| **Lower level** | LVP | Carpet tile / broadloom | Cheap thin vinyl sheet | Rugs in rec area |

## 4. Stair-specific alternatives (carpet uncertainty)

Baseline remains **carpeted treads** per [Stairs_Half_Story.md](../Architecture/Stairs_Half_Story.md). If declining full carpet:

| Rank | System | Comfort | Cleanability | Grandma look | Notes |
|---:|---|---|---|---|---|
| 1 | **Wood treads + center carpet runner** | High | High (runner replaceable) | Excellent | Best compromise; runner 27–32" wide, rod or open edges |
| 2 | **Full stair carpet** | Highest | Medium | Excellent | Quiet between levels |
| 3 | **Stained wood treads + painted risers** | Medium | High | Very good | Add non-slip tape/nosing if glossy |
| 4 | **LVP stair treads (manufacturer kit)** | Medium | High | Good if matched | Only with stair-specific nosing system |
| 5 | **Full tile stairs** | Low | High | Poor fit here | Not recommended for this aesthetic |

**Recommendation if unsure about carpet:** Choose **engineered hardwood on main floor** + **wood stairs with a traditional patterned runner**. Softness where feet land most; no wall-to-wall commitment.

## 5. Transitions & heights (Revit / detailing)

| Transition | Detail |
|---|---|
| Kitchen ↔ dining | **None** — same material continuous |
| Dining ↔ living | Same material continuous preferred |
| Dining ↔ porch | Waterproof threshold; tile/composite beyond; max step **1-1/2"–2"** if any |
| Hall ↔ stair | Carpet start at first riser or bullnose; or runner begins first tread |
| Thickness change | Reducer or T-mold only when required; prefer matched heights |

Document finish floor elevations in Revit so porch and main FF stay aligned for accessibility.

## 6. Color & material rules of thumb

**Do**

- Warm oaks, honey, soft walnut, creamy painted risers  
- Matte finishes (hide wear, less glare with skylights)  
- Area rugs with traditional pattern (Persian-inspired, floral, stripe) in living/dining  

**Don’t**

- Cool gray wood-look fad floors  
- High-gloss slippery finishes on stairs  
- Wall-to-wall carpet in kitchen or three-season porch  
- Too many different floor species on one open main level
